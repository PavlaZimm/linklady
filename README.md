
# Linklady.cz - Pavla Zimmermannová

Moderní webová prezentace pro online marketing specialistku Pavlu Zimmermannovou. Web je postaven na Next.js s čistým, responzivním designem v fialovo-žluté barevné paletě.

## Funkce webu

### Hlavní stránky
- **Home** (`/`) - Hlavní stránka s hero sekcí, službami, o mně a kontaktním formulářem
- **O mně** (`/o-mne`) - Detailní informace o Pavle, její zkušenosti a vzdělávání
- **Blog** (`/blog`) - Připravovaná sekce pro články o online marketingu
- **Kontakt** (`/kontakt`) - Kontaktní formulář a informace

### Služby
- **SEO** - Strategické SEO a lokální optimalizace
- **PPC** - Google Ads a Sklik kampaně
- **AI agenti** - Chytré nástroje pro analýzu a optimalizaci
- **Automatizace** - Automatizace rutinních úkolů

### Technické vlastnosti
- Next.js 15 s App Router
- TypeScript
- Tailwind CSS s custom fialovo-žlutou paletou
- Responzivní design
- SEO optimalizované (metadata.json)
- Moderní komponenty a navigace
- **Plynulé fade-in animace při scrollování** - Intersection Observer API pro detekci prvků

### Design
- **Barvy**: Fialová (#7B1FA2) jako primární, žlutá (#FFC107) jako sekundární
- **Typografie**: Geist Sans a Geist Mono fonty
- **Layout**: Čistý, moderní design s důrazem na konverze
- **Komponenty**: Karty služeb, hero sekce, kontaktní formuláře

## Struktura projektu

```
app/
├── page.tsx          # Hlavní stránka
├── o-mne/page.tsx    # O mně stránka
├── blog/page.tsx     # Blog (připravuje se)
├── kontakt/page.tsx  # Kontakt
├── layout.tsx        # Root layout
├── globals.css       # Globální styly
└── metadata.json     # SEO metadata

components/
└── navigation.tsx    # Hlavní navigace
```
